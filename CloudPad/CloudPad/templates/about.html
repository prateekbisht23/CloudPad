About CloudPad

Welcome to CloudPad — a lightweight, real-time text and file-sharing platform designed to make quick sharing effortless and secure. Built with simplicity and efficiency in mind, CloudPad lets you create a unique URL where you can instantly share text and files without the hassle of registration or logins.

👨‍💻 About the Creator

Hi! I’m Prateek Bisht, a passionate software developer and a final-year computer science student. My journey into web development started with a curiosity for how things work behind the scenes — from dynamic websites to AI-driven applications.

CloudPad is the result of combining my passion for building real-world solutions with my technical expertise. I wanted to create something that works instantly, with no friction — just a fast and reliable sharing experience.

💡 Inspiration Behind CloudPad

CloudPad was inspired by platforms like Dontpad, which offer quick text sharing. However, I wanted to take it a step further by adding file-sharing capabilities while maintaining a minimalist and intuitive interface.

  • No registration required
  • Instant text and file-sharing
  • Secure and temporary storage
  • Simple and clean UI

🧩 Challenges I Faced

  • Real-Time Syncing: Syncing file uploads and text changes was tricky, but adjusting the Supabase setup fixed it.
  • File Handling: Uploading files and setting permissions on Supabase required fine-tuning.
  • UI Consistency: Making the UI responsive across devices took several iterations.
  • Bucket Permissions: Fixing Supabase permissions ensured secure and consistent access.
